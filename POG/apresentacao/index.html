RCP Pattern ( Reuse by Copy-and-Paste )

  • Se precisar alterar, copie e cole de novo!

User Friendly Exception

Padronização de todas as mensagens de erro para uma única mensagem amigável Dê preferência a uma mensagem que coloque a culpa no usuário 

                        public static void main(String[] args) {
    while (true) {
        try {
        // ...
        } catch (Throwable ex) {
            System.out.println("Caro usuário, tente novamente"          "observando as regras de uso do sistema");
            // após a mensagem, o while(true) garante a
            //robustez do sistema que não aborta nunca!
        }
    }
}
