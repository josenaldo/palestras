Como criar uma aplicação

Crie uma conta no Openshift

Acesse https://www.openshift.com/ e crie sua conta.

Instale o RHC

Acesse https://www.openshift.com/developers/rhc-client-tools-install e veja como instalar o RHC.

Configure o RHC

ATENÇÃO! A EXECUÇÃO DOS COMANDOS ABAIXO DEVEM SER FEITAS NA ORDEM EXATA!



							$ rhc setup

Tempo estimado de execução: 363526532632 horas.

Crie sua aplicação



							$ rhc app create app_name web_language

Se eu quiser fazer Hot deploy?

Ao executar o build, o cartridge para e volta após o deploy.

para mudar para Hot deploy, basta inserir um marker file


					$ touch .openshift/markers/hot_deploy
$ git add .openshift/markers/hot_deploy
$ git commit -m "Changing application to hot deploy"
$ git push

Basta renomear o arquivo e executar um novo push reiniciar o servidor
