Introdução ao Openshift

IAAS

Infraestrutura como Serviço

SAAS

Software como Serviço

PASS

Plataforma como Serviço

IaaS

IaaS
Na Infraestutura como Serviço, o provedor oferece hardware virtual para o cliente. Esse hardware é pré-configurado e pode ser customizado de acordo com as necessidades do cliente. O cliente é responsável por instalar todo software necessário e criar sua aplicação. É a opção mais flexível. Normalmente, é focada em sysadmins. Exemplos: Amazon EC2, Microsoft Azure e Rackspace.
SaaS
No Software como Serviço, o provedor oferece o software pronto para o cliente. Esse software pode ter algumas opções de customização, mas elas são muito limitadas.
PaaS

Versões do Openshift

Open shift Origin é o projeto do Openshift. É a origem das outras duas versões e está disponível no http://openshift.github.io/. Se você quiser usar essa versão, vai ter de instalá-la em sua própria infraestutura.

O Openshift Online é a versão é o Openshift Origin instalado e gerenciado pela Red Hat. Você só precisa criar uma conta e se concentrar em criar o seu projeto. Uma nova versão é liberada a cada 3 semanas (um sprint).

O Openshift Enterprise é uma solução completa, voltada para empresas. Seu foco é estabilidade e pode ser instalada em seu próprio datacenter ou uem um provedor IaaS, como o AWS ou Rackspace. Vem com o Openshift e o Red Hat Enterprise Linux. A cada trimestre, é liberada uma nova versão.

Passos para criar uma aplicação

AVISO! ESSE PROCESSO É COMPLICADO, EXTREMAMENTE DEMORADO E FRUSTRANTE!

Crie uma conta no Openshift

Acesse https://www.openshift.com/ e crie sua conta.

Tempo estimado: 5 dias.

Instale o RHC

Acesse https://www.openshift.com/developers/rhc-client-tools-install e veja como instalar o RHC.

Tempo estimado: 45 dias.

Configure o RHC

ATENÇÃO! A EXECUÇÃO DOS COMANDOS ABAIXO DEVEM SER FEITAS NA ORDEM EXATA!

							
								$ rhc setup

Tempo estimado: 4 anos e 78 dias.

Alguns comandos úteis:

Realiza o setup do RHC para uma outra conta


								rhc setup -l outroemail@seuservidor.com

Faz logout da sua conta


								rhc account logout

Ver informações úteis sobre a aplicação


								rhc show-app gambiarra

Finalmente, crie sua aplicação

SE VOCÊ CONSEGUIU CHEGAR ATÉ AQUI, PARABÉNS! AGORA, EXECUTE ESSA SÉRIE DE COMANDOS...

							
								$ rhc app create app_name web_language

Tempo estimado: DESCONHECIDO.

Você também pode usar o console web!

https://openshift.redhat.com/app/console/applications
